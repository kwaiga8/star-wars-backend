## Description

Simple STAR WARS API backend with NEST for training

## Open Api Documentation

```bash
http://localhost:3000/documentation/
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode 
$ npm run start:prod

# database migration
$ npm run db:migrate
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
